# Errbit-NG

This is fork of [errbit](https://github.com/errbit/errbit).

## Tech Stack

* Ruby 3.1 (for JRuby support), 3.2 and 3.3
* Ruby on Rails

## Features

* Modern Ruby 3.1 or later
* Modern Ruby on Rails 6.1. 7.0+ upgrade in progress
* JRuby support (in progress)
